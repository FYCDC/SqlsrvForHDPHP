SqlsrvForHDPHP
==============

HDPHP 微软Sqlsrv 支持库


名称：hdphp 操作mssql 类
文件：DbSqlsrv.class.php

使用：
配置文件config.php
	'DB_DRIVER'                     => 'Sqlsrv',    //数据库驱动
   
	'DB_CHARSET'                    => 'utf-8',      //数据库字符集
    
	'DB_HOST'                       => '127.0.0.1', //数据库连接主机  如127.0.0.1
    
	'DB_PORT'                       => 1433,        //数据库连接端口
    
	'DB_USER'                       => 'sa',      //数据库用户名
    
	'DB_PASSWORD'                   => 'sa',          //数据库密码
    
	'DB_DATABASE'                   => 'EofficeII',          //数据库名称


使用方法看hdphp手册
